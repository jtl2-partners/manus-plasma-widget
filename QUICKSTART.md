# Quick Start Guide - Manus Monitor Widget

Get up and running with the Manus Monitor widget in 5 minutes!

## Step 1: Install the Widget

Run the installation script:
```bash
cd manus-plasma-widget
./install.sh
```

## Step 2: Add to Your Desktop or Panel

1. Right-click on your desktop or panel
2. Click **"Add Widgets..."**
3. Search for **"Manus Monitor"**
4. Drag it to your desktop or panel

## Step 3: Configure Your API Key

1. Right-click on the widget
2. Select **"Configure Manus Monitor..."**
3. Enter your Manus API key
4. Click **"OK"**

## Step 4: Enjoy!

The widget will now display:
- ✅ System health status (color-coded)
- 📊 Credits spent this month
- 💰 Credits remaining

The data refreshes automatically every 5 minutes.

## Getting Your API Key

To obtain your Manus API key:
1. Visit https://manus.im
2. Log in to your account
3. Navigate to Settings → API Keys
4. Generate a new API key
5. Copy and paste it into the widget configuration

## Tips

- **Panel Mode**: Shows a compact icon with health indicator
- **Desktop Mode**: Shows full dashboard with all metrics
- **Manual Refresh**: Click the refresh button to update immediately
- **Customize**: Toggle which metrics to display in the configuration
